---
name: AP style — no Oxford commas
description: NapaServe and Napa Valley Features follow AP style; never use Oxford commas in copy
type: feedback
---

All user-facing copy in NapaServe and Napa Valley Features must follow AP style — specifically, no Oxford (serial) commas.

**Why:** Editorial standard the user enforces across the platform and media brand.

**How to apply:** When writing or editing any display text, prose, or labels in the codebase, omit the comma before the conjunction in a series of three or more items (e.g., "A, B and C" not "A, B, and C").
